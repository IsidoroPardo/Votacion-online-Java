VOTACIONES ONLINE CON JAVA WEB

Una aplicación web con la que crear, gestionar y analizar elecciones hecha con Java Web, Servlets y JSP. Gestión de perfiles de entre los que se encuentran usuario, analista y administrador. Cada uno con su propio menú y funciones reservadas. 

Características. 
- Inicio de sesión 
- Tres perfiles diferenciados
- Encriptación de contraseñas 
- Manejo y modificación de bases de datos 
- Modelo vista controlador

Tecnologías usadas:
- Java Web
- Servlets
- JSP
- SQL

Configuración de la Base de Datos:
1. Crea la base de datos en MySQL ejecutando los archivos en la carpeta sql:
- 01_schema.sql
- 02_data.sql

2. Verifica que MySQL esté corriendo en:
- Host: localhost
- Puerto: 3306

3. Configura la ruta en src/main/resurces/db.properties
